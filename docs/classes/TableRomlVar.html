Class TableRomlVar

A RoML variable which holds a table.

Events

TableRomlVar.Changed
The Changed Event. Observers of this Event are notified whenever the __newindex metamethod is called. Observing functions are passed three arguments, the table key, the old value, and the new value.

Methods

TableRomlVar.new (self[, t={}])
Create a table variable with the specified starting table.

Parameters:

  • self TableRomlVar
  • t table The starting table. (default {})

Metamethods

TableRomlVar.__newindex (self, key, value)
Set the value for a key. The key can be anything except for the strings Changed, and _table (a private member used to store the actual table values). All observers of the Changed Event are notified when this method is called.

Parameters:

  • self TableRomlVar
  • key The key to associate the table value with.
  • value The value to place in the table.
