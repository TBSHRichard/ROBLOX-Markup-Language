Class GenericRomlVar

A RoML variable that holds a generic data type value (string, number, boolean).

Info:

  • Author: Richard Voelker

Events

GenericRomlVar.Changed
The Changed Event. Observers of this Event are notified whenever the Set method is called. Observing functions are passed two arguments, the old value and the new value.

Methods

GenericRomlVar.new (self[, value])
Create a new GenericRomlVar with the specified value.

Parameters:

  • self GenericRomlVar
  • value generic The value to set to. (optional)
GenericRomlVar.Set (self, value)
Set the value for this object. It must be another generic data type and cannot be a table. All observers of the Changed Event are notified.

Parameters:

  • self GenericRomlVar
  • value generic The value to set to.
